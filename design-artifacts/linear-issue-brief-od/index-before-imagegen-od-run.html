Design note

This direction is an operational ledger with a preservation-office tone: warm paper, ink rules, serif hierarchy, compact metadata, and restrained status color. It avoids dashboard chrome and decorative cards; the visual affordance is the repeatable relationship between the left rail, the numbered issue fields, and the Markdown headings Linear receives.
